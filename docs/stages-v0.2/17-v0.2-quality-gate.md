# Stage 17 — v0.2 Quality Gate

## 目标

完成 v0.2 的最终质量门禁，确认项目可以打 tag `v0.2.0`。

本阶段不新增大功能，只做修复、回归、文档一致性和最终验收。

## 检查范围

### 后端

- loader。
- symbol extraction。
- call extraction。
- interface implementation expansion。
- graph filtering。
- HTTP API。
- source and callsite APIs。
- metadata and rescan。

### 前端

- symbol search。
- package filter。
- graph filter toggles。
- React Flow layout。
- node source。
- edge callsite。
- warnings。
- meta/rescan panel。

### 工程

- Makefile。
- smoke scripts。
- README。
- demo docs。
- CI workflow。
- release scripts。

## 必跑命令

```bash
git status --short
make check
make web-build
make build
make release
./scripts/smoke.sh
```

启动 demo：

```bash
./bin/codemap serve ./examples/layered-service --port 8080
```

另开终端：

```bash
curl -s http://localhost:8080/api/health
curl -s http://localhost:8080/api/meta | python -m json.tool
curl -s "http://localhost:8080/api/graph?entry=main.main&depth=5" | python -m json.tool
curl -s "http://localhost:8080/api/graph?entry=main.main&depth=5&show_external=true" | python -m json.tool
curl -s "http://localhost:8080/api/graph?entry=main.main&depth=5&expand_interface=true" | python -m json.tool
```

如果支持 callsite：

```bash
curl -s "http://localhost:8080/api/callsite?edge_id=<real_edge_id>" | python -m json.tool
```

## 人工验收

浏览器打开：

```text
http://localhost:8080
```

检查：

1. Web UI 首页可打开。
2. 可搜索 `main` 并加载 graph。
3. depth 控制生效。
4. package filter 生效。
5. show external toggle 生效。
6. show interface / expand interface 生效。
7. 点击 node 显示 source。
8. 点击 edge 显示 callsite。
9. warnings 显示正常。
10. meta 信息显示正常。
11. rescan 按钮可用。
12. API 错误时 UI 不白屏。

## 文档一致性检查

确认 README 没有虚假声明：

- 不支持远程 GitHub URL。
- 不默认分析 `_test.go`。
- interface expansion 是静态候选，不是运行时真实调用链。
- 不包含 LLM。
- 不包含 IDE 插件。
- 不包含数据库。

## Git 检查

不应提交：

```text
bin/
dist/
web/node_modules/
coverage files
local env files
```

可以提交：

```text
internal/server/static/
docs/demo/*.png
```

前提是 README 明确说明构建流程。

## v0.2 已知限制模板

README 可保留：

```text
Known limitations:
- Only local Go repositories are supported.
- Test files are excluded by default.
- Interface expansion shows static candidates, not guaranteed runtime dispatch.
- Dynamic function values and reflection are reported conservatively.
- Field-level read/write references are not implemented.
- No LLM explanation layer is included.
```
